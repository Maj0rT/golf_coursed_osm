# Golfplatz-Geodaten

Geodaten der Golfplätze der App **Grünlage Golf** — je Platz eine Datei
`courses/<slug>.json` mit der Geometrie (Fairways, Grüns, Bunker,
Abschläge, Wasser, Spiellinien).

Aktuell **462 Plätze**.

Die Dateien werden automatisch vom Collector-Server veröffentlicht: bei
jeder Änderung eines Platzes, beim Akzeptieren einer eingereichten
Korrektur und per Voll-Export. Quelle sind OpenStreetMap sowie von
Nutzern eingereichte Ergänzungen. Nicht manuell bearbeiten — Änderungen
werden beim nächsten Export überschrieben.
